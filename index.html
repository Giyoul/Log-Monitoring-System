리소스 병목 순위

  1. CRITICAL
    물리 메모리 (RAM)

    Unused 최솟값 68 MB — 16 GB 전체에서 잔여 0.4%. 메모리 스파이크 하나로 즉시 스왑 압력 발생. 현재 스왑아웃이 없는 것은 Apple Unified Memory의 고속 NVMe 스왑 덕분이나, 오디오 버퍼 지연·드롭아웃 위험은 상존.

  2. HIGH
    E-Cluster (효율 코어)

    평균 Active Residency 80.9%. 백그라운드 OS 작업·플러그인 스캔·네트워크 I/O가 효율 코어를 지속 점유. P-코어가 오디오 처리에 집중할 여유가 줄어드는 간접 압박.

  3. HIGH
    P-Cluster 클럭 포화

    평균 3,099 MHz · 최대 3,111 MHz (M1 Max spec). 성능 코어가 3시간 내내 최고 클럭을 유지 — 추가 CPU 여유 없이 빡빡하게 운용 중.

  4. LOW
    디스크 I/O 순간 피크

    평균 42 MB/s로 안정적이나 p95 193 MB/s, 최대 275 MB/s 순간 피크 발생. 현행 NVMe SSD 한계에서 여유가 있어 크리티컬하지 않음.
